# GAS(Google Apps Script)連携のセットアップ

一部のアプリは、クラス全体の回答をスプレッドシートに集めるための **GAS連携(任意)** に対応しています。
GASを設定しなくても、すべてのアプリは端末内保存だけで動作します。

| アプリ | GASコード | 連携内容 |
|---|---|---|
| 3年 すきなものアンケート&グラフ | `survey.gs` | 回答(番号・カテゴリ・項目・すき)を1行ずつ記録 |

## 共通セットアップ手順

1. Googleスプレッドシートを新規作成
2. 「拡張機能 → Apps Script」を開き、対応する `.gs` ファイルの中身を貼り付けて保存
3. 「デプロイ → 新しいデプロイ → 種類:ウェブアプリ」
   - **次のユーザーとして実行**:自分
   - **アクセスできるユーザー**:**「組織内の全員」**(※「全員」にしないこと。児童のデータを扱うため必ず組織内限定で運用してください)
4. 発行された WebアプリURL(`https://script.google.com/macros/s/…/exec`)をコピー
5. アプリ内の設定欄(例:すきなものアンケートの「せんせい用グラフ」画面)に貼り付け

## 運用上の注意

- 記録するのは出席番号ベースのデータのみとし、氏名は扱わない設計にしています
- スプレッドシートの共有範囲も組織内限定にしてください
- 年度が変わったらシートを新しくする(またはログを消す)ことをおすすめします
