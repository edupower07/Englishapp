# Picture Dictionary 機能まるわかりムービー 構成台本

**ファイル**: `promo/pd_feature_movie.mp4`(1280x720 / 30fps / 1分48秒 / アプリ内音声+BGM・字幕形式)

操作デモはすべて **実際のPicture Dictionary(ローカル起動した本体)を操作した実演映像** で、
**1.5倍速** で再生してテンポよく紹介しています(タイトル・クロージングは等速)。

**アプリ内の読み上げ音声も収録しています。** 録画時に `speak()`(Web Speech API)の
呼び出し(単語・速度・タイミング)をすべて記録し、同じ内容を音声合成して正確な位置に
配置しました(カードタップの発音、🐢ゆっくり音声、クイズの "Good job!"、かるたの
"Ready? Go!" / "Yes!"、ビンゴの "Bingo!"、スピーチの読み上げなど)。声が鳴っている間は
BGMが自動で小さくなります。声は等速のまま配置しているので聞き取りやすくなっています。

## 制作メモ

- PictureDictionary リポジトリをローカルで配信し、絵カード画像も**実物**を表示して録画しています。
- 🎤発音練習の録音デモは Chromium のフェイクマイクデバイスで実際に録音→再生しています。
- BGMはスクリプト合成のオリジナル曲(権利フリー)。ナレーションは字幕形式です。
- 再生成方法は `promo/tools/README.md` を参照(PD用は `record/pd_segments.js` → `record/build_pd.py`)。
